# E-commerce Backend API (Node.js + PostgreSQL)

This is the backend of a full-stack e-commerce platform built with Node.js, Express, and PostgreSQL using Sequelize ORM. It provides a secure, scalable RESTful API.

## Features

- JWT-based user authentication
- Role-based access control (admin, user)
- Product and category management
- Shopping cart and order processing
- Input validation and error handling
- Scalable architecture

## Stack

- Node.js
- Express
- PostgreSQL
- Sequelize
- JWT
- dotenv

## Frontend

[final-project-nodejs-react-modules-frontend](https://github.com/serg-backend-developer/final-project-nodejs-react-modules-frontend)

## Getting Started

```bash
npm install
npx sequelize db:migrate
npx sequelize db:seed:all
